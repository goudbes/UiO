# INF2100 - Programming project (UiO, Autumn 2016)

The assignment was to build a compiler implemented in Java for a mini-version of Pascal called Pascal2016. The whole project was divided into four deliveries:
- Scanner 
- Parser 
- Checker 
- Code generator
